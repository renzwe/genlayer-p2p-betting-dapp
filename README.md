# 🚀 From Zero to GenLayer — P2P Betting dApp

This project is a beginner-friendly tutorial for building a simple dApp on GenLayer.

## 🧠 What You’ll Learn

- Optimistic Democracy Consensus
- Equivalence Principle
- Python Intelligent Contracts
- Frontend interaction with genlayer-js

## 🛠 Project Overview

A simple P2P betting app where:
- Users create bets
- Others join
- Results are submitted
- Validators decide the outcome

## 📂 Project Structure

- contracts/ → Python Intelligent Contract
- frontend/ → Example interaction using genlayer-js

## ⚙️ How to Run

```bash
npm install
node frontend/app.js
